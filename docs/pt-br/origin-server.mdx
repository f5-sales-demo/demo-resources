O servidor de origem é uma VM Azure que hospeda aplicações web vulneráveis atrás
de um proxy reverso nginx. Ele fornece alvos para demonstrar os recursos de segurança
do F5 Distributed Cloud.

## Funcionalidades do F5 XC Validadas

- **Web Application Firewall** — injeção SQL, XSS, injeção de comandos, travessia de diretórios
- **Segurança de API** — vulnerabilidades de API REST e GraphQL, OWASP API Top 10
- **Defesa contra Bots** — alvos de credential stuffing, endpoints de scraping
- **Defesa do Lado do Cliente** — injeção de JavaScript, ataques estilo Magecart

Consulte a [documentação do Servidor de Origem](https://f5xc-salesdemos.github.io/origin-server/)
para arquitetura, aplicações, variáveis do terraform e detalhes de implantação.
