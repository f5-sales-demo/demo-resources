Il generatore di traffico è una VM Azure che produce traffico di attacco, scansioni
di ricognizione e simulazione di bot contro i bilanciatori di carico HTTP di F5 Distributed Cloud. Verifica
che le policy WAF, Bot Defense, API Security e Client-Side Defense
siano configurate correttamente.

## Funzionalità F5 XC Validate

- **Web Application Firewall** — payload di SQL injection, XSS, command injection, path traversal
- **Bot Defense** — automazione con browser headless, credential stuffing, crawling rapido
- **API Security** — attacchi OWASP API Top 10, abuso di GraphQL, fuzzing dei parametri
- **Client-Side Defense** — iniezione di skimmer in stile Magecart, manipolazione del DOM

Consultare la [documentazione del Generatore di Traffico](https://f5xc-salesdemos.github.io/traffic-generator/)
per il catalogo degli strumenti, le suite di traffico, le variabili terraform e i dettagli di distribuzione.
