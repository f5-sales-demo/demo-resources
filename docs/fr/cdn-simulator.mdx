Le simulateur CDN est une VM Azure exécutant NGINX configurée pour se comporter comme un
nœud edge CDN. Il injecte des en-têtes réalistes spécifiques aux fournisseurs et simule le
comportement de cache pour les démonstrations d'intégration CDN avec F5 Distributed Cloud.

## Fournisseurs simulés

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consultez la [documentation du simulateur CDN](https://f5xc-salesdemos.github.io/cdn-simulator/)
pour la configuration nginx, les en-têtes des fournisseurs, les variables terraform et les détails de déploiement.
