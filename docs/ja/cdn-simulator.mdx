CDNシミュレーターは、CDNエッジノードのように動作するよう構成されたNGINXを実行するAzure VMです。リアルなベンダー固有のヘッダーを注入し、F5 Distributed CloudとのCDN統合デモ用にキャッシュ動作をシミュレートします。

## シミュレート対象ベンダー

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

nginx構成、ベンダーヘッダー、Terraform変数、およびデプロイの詳細については、[CDNシミュレータードキュメント](https://f5xc-salesdemos.github.io/cdn-simulator/)を参照してください。
