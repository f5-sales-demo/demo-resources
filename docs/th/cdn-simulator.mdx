ตัวจำลอง CDN เป็น Azure VM ที่รัน NGINX ซึ่งได้รับการกำหนดค่าให้ทำงานเหมือน CDN
edge node โดยจะแทรก header เฉพาะผู้ให้บริการที่สมจริง และจำลองพฤติกรรมแคช
สำหรับการสาธิตการผสานรวม CDN กับ F5 Distributed Cloud

## ผู้ให้บริการที่จำลอง

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

ดู[เอกสารประกอบตัวจำลอง CDN](https://f5xc-salesdemos.github.io/cdn-simulator/)
สำหรับการกำหนดค่า nginx, header ของผู้ให้บริการ, ตัวแปร terraform และรายละเอียดการ deploy
