เซิร์ฟเวอร์ต้นทางเป็น Azure VM ที่โฮสต์เว็บแอปพลิเคชันที่มีช่องโหว่อยู่เบื้องหลัง
nginx reverse proxy โดยให้เป้าหมายสำหรับสาธิตฟีเจอร์ด้านความปลอดภัยของ F5 Distributed
Cloud

## ฟีเจอร์ F5 XC ที่ได้รับการตรวจสอบ

- **Web Application Firewall** — SQL injection, XSS, command injection, path traversal
- **ความปลอดภัย API** — ช่องโหว่ของ REST และ GraphQL API, OWASP API Top 10
- **การป้องกันบอท** — เป้าหมายสำหรับ credential stuffing, endpoint สำหรับการ scraping
- **การป้องกันฝั่งไคลเอนต์** — JavaScript injection, การโจมตีแบบ Magecart
