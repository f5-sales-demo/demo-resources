เครื่องสร้างทราฟฟิกเป็น Azure VM ที่สร้างทราฟฟิกโจมตี การสแกนสำรวจ
และการจำลองบอต เพื่อทดสอบ HTTP load balancers ของ F5 Distributed Cloud โดย
ใช้ตรวจสอบว่านโยบาย WAF, Bot Defense, API Security และ Client-Side Defense
ได้รับการกำหนดค่าอย่างถูกต้อง

## ฟีเจอร์ F5 XC ที่ได้รับการตรวจสอบ

- **Web Application Firewall** — เพย์โหลด SQL injection, XSS, command injection, path traversal
- **Bot Defense** — การทำงานอัตโนมัติด้วย headless browser, credential stuffing, การ crawl อย่างรวดเร็ว
- **API Security** — การโจมตี OWASP API Top 10, การใช้งาน GraphQL ในทางที่ผิด, parameter fuzzing
- **Client-Side Defense** — การแทรก skimmer แบบ Magecart, การจัดการ DOM

ดู[เอกสารเครื่องสร้างทราฟฟิก](https://f5xc-salesdemos.github.io/traffic-generator/)
สำหรับรายการเครื่องมือ ชุดทราฟฟิก ตัวแปร terraform และรายละเอียดการติดตั้ง
