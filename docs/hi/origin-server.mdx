ऑरिजिन सर्वर एक Azure VM है जो nginx रिवर्स प्रॉक्सी के पीछे कमजोर वेब एप्लिकेशन होस्ट करती है। यह F5 Distributed Cloud सुरक्षा सुविधाओं को प्रदर्शित करने के लिए लक्ष्य प्रदान करती है।

## F5 XC सुविधाएँ जो मान्य की गईं

- **वेब एप्लिकेशन फ़ायरवॉल** — SQL इंजेक्शन, XSS, कमांड इंजेक्शन, पाथ ट्रैवर्सल
- **API सुरक्षा** — REST और GraphQL API कमजोरियाँ, OWASP API Top 10
- **बॉट डिफेंस** — क्रेडेंशियल स्टफिंग लक्ष्य, स्क्रैपिंग एंडपॉइंट
- **क्लाइंट-साइड डिफेंस** — JavaScript इंजेक्शन, Magecart-शैली के हमले

आर्किटेक्चर, एप्लिकेशन, terraform वेरिएबल्स, और डिप्लॉयमेंट विवरण के लिए [ऑरिजिन सर्वर दस्तावेज़ीकरण](https://f5xc-salesdemos.github.io/origin-server/) देखें।
