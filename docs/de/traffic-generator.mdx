Der Traffic-Generator ist eine Azure-VM, die Angriffsverkehr, Aufklärungsscans und Bot-Simulationen gegen F5 Distributed Cloud HTTP-Load-Balancer erzeugt. Er überprüft, ob WAF-Richtlinien, Bot Defense, API Security und Client-Side Defense korrekt konfiguriert sind.

## Validierte F5 XC-Funktionen

- **Web Application Firewall** — SQL-Injection, XSS, Command-Injection, Path-Traversal-Payloads
- **Bot Defense** — Headless-Browser-Automatisierung, Credential Stuffing, schnelles Crawling
- **API Security** — OWASP API Top 10-Angriffe, GraphQL-Missbrauch, Parameter-Fuzzing
- **Client-Side Defense** — Magecart-artige Skimmer-Injection, DOM-Manipulation

Weitere Informationen finden Sie in der [Traffic-Generator-Dokumentation](https://f5xc-salesdemos.github.io/traffic-generator/) mit Tool-Katalog, Traffic-Suiten, Terraform-Variablen und Bereitstellungsdetails.
