Der Ursprungsserver ist eine Azure-VM, die verwundbare Webanwendungen hinter einem nginx-Reverse-Proxy hostet. Er stellt Ziele zur Demonstration der Sicherheitsfunktionen von F5 Distributed Cloud bereit.

## Validierte F5 XC-Funktionen

- **Web Application Firewall** — SQL-Injection, XSS, Command-Injection, Path-Traversal
- **API-Sicherheit** — REST- und GraphQL-API-Schwachstellen, OWASP API Top 10
- **Bot-Abwehr** — Credential-Stuffing-Ziele, Scraping-Endpunkte
- **Client-seitiger Schutz** — JavaScript-Injection, Magecart-artige Angriffe

Weitere Informationen zu Architektur, Anwendungen, Terraform-Variablen und Bereitstellungsdetails finden Sie in der [Dokumentation des Ursprungsservers](https://f5xc-salesdemos.github.io/origin-server/).
