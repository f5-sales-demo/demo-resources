El generador de tráfico es una VM de Azure que produce tráfico de ataque, escaneos
de reconocimiento y simulación de bots contra los balanceadores de carga HTTP de F5 Distributed Cloud. Valida
que las políticas de WAF, Bot Defense, API Security y Client-Side Defense
estén configuradas correctamente.

## Características de F5 XC Validadas

- **Web Application Firewall** — cargas útiles de inyección SQL, XSS, inyección de comandos y traversal de rutas
- **Bot Defense** — automatización de navegadores headless, credential stuffing, rastreo rápido
- **API Security** — ataques OWASP API Top 10, abuso de GraphQL, fuzzing de parámetros
- **Client-Side Defense** — inyección de skimmers estilo Magecart, manipulación del DOM

Consulte la [documentación del Generador de Tráfico](https://f5xc-salesdemos.github.io/traffic-generator/)
para el catálogo de herramientas, suites de tráfico, variables de terraform y detalles de despliegue.
