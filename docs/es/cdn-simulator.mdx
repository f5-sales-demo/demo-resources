El simulador de CDN es una VM de Azure que ejecuta NGINX configurada para comportarse como un
nodo de borde de CDN. Inyecta encabezados realistas específicos de cada proveedor y simula el comportamiento
de caché para demostraciones de integración de CDN con F5 Distributed Cloud.

## Proveedores Simulados

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

Consulte la [documentación del Simulador de CDN](https://f5xc-salesdemos.github.io/cdn-simulator/)
para obtener información sobre la configuración de nginx, encabezados de proveedores, variables de terraform y detalles de despliegue.
