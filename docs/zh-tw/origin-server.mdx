來源伺服器是一台 Azure VM，在 nginx 反向代理後方託管易受攻擊的網頁應用程式。它提供用於展示 F5 Distributed Cloud 安全功能的目標。

## 已驗證的 F5 XC 功能

- **Web 應用程式防火牆** — SQL 注入、XSS、命令注入、路徑遍歷
- **API 安全** — REST 和 GraphQL API 漏洞、OWASP API Top 10
- **Bot 防禦** — 憑證填充目標、爬取端點
- **用戶端防禦** — JavaScript 注入、Magecart 風格攻擊

請參閱[來源伺服器文件](https://f5xc-salesdemos.github.io/origin-server/)以了解架構、應用程式、terraform 變數和部署詳細資訊。
