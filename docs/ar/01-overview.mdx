موارد العرض التوضيحي هي أجهزة افتراضية مُعدّة مسبقاً في Azure يمكن نشرها عبر Terraform.
كل مكون عبارة عن جهاز افتراضي مستقل مع برامج مُصممة خصيصاً ومثبتة ومُهيأة
من خلال cloud-init عند أول إقلاع.

## اختيار المكونات

المكونات هي وحدات بناء. اختر منها بناءً على ما يحتاجه العرض التوضيحي الخاص بك:

- **خادم الأصل** يوفر تطبيقات ويب ضعيفة الحماية تعمل كأهداف لميزات
  أمان F5 XC. تحتاج معظم العروض التوضيحية إلى هذا المكون.
- **مولّد حركة المرور** ينتج حركة مرور هجومية وشرعية ضد بيئة العرض التوضيحي
  الخاصة بك. مطلوب عند عرض فعالية WAF أو اكتشاف الروبوتات أو حماية
  واجهات برمجة التطبيقات.
- **محاكي CDN** يضع حافة CDN قائمة على NGINX بين العملاء والأصل،
  ويحقن ترويسات خاصة بالمزوّد. يُستخدم لعروض تكامل CDN التوضيحية.

## المتطلبات الأساسية المشتركة

تتطلب جميع المكونات:

- اشتراك Azure مع صلاحية مساهم
- Terraform 1.5 أو أحدث
- Azure CLI مع مصادقة تم إجراؤها (`az login`)
- زوج مفاتيح SSH (المسار الافتراضي `~/.ssh/id_ed25519.pub`)

## نمط النشر

يتبع كل مكون نفس سير عمل Terraform:

1. استنساخ مستودع المكون
2. `cd terraform/`
3. نسخ `terraform.tfvars.example` إلى `terraform.tfvars` وتعيين المتغيرات المطلوبة (`subscription_id` مطلوب لجميع المكونات؛ بعض المكونات لديها مدخلات إضافية مطلوبة)
4. `terraform init && terraform plan && terraform apply`
5. التحقق عبر نقطة فحص صحة المكون

يتم تحديد معرّف الناشر تلقائياً من حساب Azure AD الخاص بك ويُضمّن في جميع أسماء الموارد (مثال: `rg-origin-server-lab-rmordasiewicz`).

## التكامل بين المكونات

تتكامل المكونات لإنشاء بيئات عرض توضيحي متكاملة:

```
Traffic Generator ──► F5 XC HTTP LB (WAF/Bot/API/CSD) ──► Origin Server
                                     ▲
                               CDN Simulator
```

**ترتيب النشر:**

1. **خادم الأصل** — يُنشر أولاً، فهو يستضيف التطبيقات التي تستهدفها جميع المكونات الأخرى
2. **موازن أحمال HTTP في F5 XC** — قم بتهيئة مجموعة أصل تشير إلى عنوان IP العام لخادم الأصل، ثم أنشئ موازن أحمال HTTP مع سياسات WAF أو الحماية من الروبوتات أو أمان واجهات برمجة التطبيقات أو الدفاع من جانب العميل
3. **مولّد حركة المرور** و/أو **محاكي CDN** — يُنشران أخيراً، مع توجيههما إلى FQDN الخاص بموازن أحمال F5 XC

تشير متغيرات Terraform الخاصة بكل مكون إلى مخرجات المكونات المنشورة مسبقاً. راجع وثائق المكون للاطلاع على متطلبات المتغيرات المحددة.

استخدم [حاسبة أسعار Azure](https://azure.microsoft.com/en-us/pricing/calculator/) للحصول على تقديرات التكلفة الحالية. قم بتشغيل `terraform destroy` عندما يكون المختبر في وضع الخمول لإيقاف الرسوم.
