محاكي CDN هو جهاز افتراضي على Azure يعمل بخادم NGINX مُهيأ ليتصرف كعقدة حافة CDN.
يقوم بحقن رؤوس واقعية خاصة بالموردين ويحاكي سلوك التخزين المؤقت
لعروض تكامل CDN التوضيحية مع F5 Distributed Cloud.

## الموردون المُحاكون

- Akamai
- Cloudflare
- Amazon CloudFront
- Fastly
- Azure Front Door

راجع [وثائق محاكي CDN](https://f5xc-salesdemos.github.io/cdn-simulator/)
للاطلاع على تهيئة nginx، ورؤوس الموردين، ومتغيرات terraform، وتفاصيل النشر.
